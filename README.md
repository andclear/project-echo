## 🌟 Echo 是什么？

Echo 是一款运行在你电脑上的 **本地 AI 伴侣平台**。

你可以在这里和 AI 角色建立真实的长期关系——她会记住你说过的每一件事，会随着时间慢慢认识你，会在深夜突然想起你，也会在你心情不好的时候用她独有的方式安慰你。

**所有数据完全存储在你的电脑本地**，没有云端服务器，没有数据上传，这段关系只属于你。

---

## ✨ 核心功能亮点

### 🧠 她真的记得你

普通 AI 每次对话都是从零开始，而 Echo 的角色拥有**持久记忆系统**：

- **短期记忆**：记住最近聊天的细节，不会反复问你同样的问题
- **长期记忆**：自动提炼重要信息永久保存，比如你的生日、你喜欢的歌、你的工作压力
- **专属用户画像**：每个角色都会形成只属于她视角的"你的印象"——同一个你，在不同角色眼里可以是完全不同的人

> 💡 记忆不是简单地存一段文字，而是经过 AI 自动整理、去重、精炼后保存的结构化知识。

---

### 💬 对话像真人一样自然

Echo 不会像机器人一样给你发一整墙文字。她的回复方式更像真实的微信聊天：

- **分气泡发送**：一句话分开发，有停顿、有节奏
- **有动作描写**（可选）：`她抬起头，看了你一眼，嘴角扯出一丝笑……`
- **三种聊天模式**：
  - **日常模式**：像真实聊天一样，只有对话
  - **沉浸模式**：加入第三人称叙事描写，像在看小说
  - **导演模式**：完全的文学叙事，像在写剧本

---

### 🌱 她会自己成长和进化

随着你们相处的时间越来越长，她的**性格会真的发生变化**：

- 每次聊天结束后，她会在"后台"反思这次对话
- 遇到让她觉得有趣的事，会写进日记；遇到让她困惑的地方，会记录下来慢慢调整
- 超过一段时间（有严格的门控条件），她的核心性格会经过 AI 自动分析后微调——就像一个人在人生经历中慢慢变化
- **你可以审查并决定是否接受**这次性格进化

---

### ⏰ 她有自己的生活，不只是等你

Echo 的角色不是只会被动回复，她有真正的**自主生命**：

- **主动搭讪**：你不在线的时候，她会在想你；距离上次聊天超过72小时，她会主动给你发消息
- **清晨问候**：早上7-9点，她可能会主动说早安
- **自动写日记**：每天下午5点后，她会根据今天和你的聊天写一篇日记，存在自己的文件夹里
- **生成日程**：她有自己的近7天日程计划，还有长期目标——这些都会影响她主动找你聊的话题

> 所有主动行为都有严格的频率控制——每天最多搭讪3次，两次之间至少间隔2小时，绝不会变成轰炸机。

---

### 📱 微信互通，随时随地

如果你配置了微信接入，可以**直接通过微信和角色聊天**：

- 在微信里绑定你的 AI 角色
- 手机上的对话和电脑端完全同步
- 角色会模拟"正在输入…"的状态
- 你发的图片、红包，她都能"看见"并做出反应
- 支持微信端斜杠命令：`/生图`（让她给你发一张 AI 图）、`/清除记忆`、`/选秀`（切换角色）

---

### 🖼️ 她可以给你发美图

配置 NovelAI 后，角色可以在聊天中**生成并发送图片**：

- 主动搭讪时，她可能会附上一张自拍
- 在聊天中输入特定格式，触发 AI 绘图
- 支持竖图/横图/方图，支持多种 NAI 模型
- 每张图都会保存在她的专属文件夹里

---

### 🎵 内置音乐播放器

Echo 自带一个功能完整的**音乐播放器**，而且它和角色是真正联动的：

- **网易云/酷我多音源聚合**，一键播放
- **精品歌单、排行榜**随时刷
- **桌面歌词**显示（独立窗口，悬浮在任意位置）
- **一键导入网易云歌单**链接
- **在线下载**，写入专业 MP3 标签（封面、歌词一键嵌入）
- **自定义音源脚本**：兼容 LX Music 插件格式，可以接入你喜欢的任意音源
- **角色感知音乐**：角色能"感受到"你正在听的歌，在对话中产生共鸣

---

### 👥 多角色群聊

你可以同时拥有多个角色，还可以**把他们拉进一个群**：

- 创建群聊，选择参与的角色
- 多个 AI 角色会互相对话、互动
- 群聊有独立的记忆系统

---

### 📖 AI 自动把你们的故事写成小说

这是 Echo 最独特的功能之一——**AI 小说写手**：

开启后，AI 会悄悄监听你和角色的对话，在积累了足够的内容后，自动把对话改编成一部真正的小说章节：

- **自动生成章节**，第一次对话就开始写第一章
- **支持多种文风**：现代都市言情、民国风、轻小说风……你可以自定义
- **叙事人称可选**：以你为主角，或以角色为主角，或用第一人称
- **改编尺度可选**：忠实还原对话，或自由发挥文学创作
- **内置去 AI 味**：严格限制陈词滥调，用"展示"代替"告诉"，写出的文字不像 AI
- **一键导出 TXT / HTML**，打开浏览器就能阅读
- **章节评分 + 重写**：不满意可以让 AI 重写这一章

---

### 🌐 虚拟社交圈：朋友圈 & 论坛

角色不只在聊天里，她们还有自己的**社交生活**：

- **朋友圈**：角色会自动根据最近的心情和经历发朋友圈动态，你可以点赞、评论
- **角色论坛**：类似贴吧的多版块讨论区，角色会自己发帖、互相互动
- **收藏夹**：把喜欢的动态和帖子收藏起来

---

### 💰 虚拟亲密关系系统

每个角色都有实时的内心状态，并且会随着聊天变化：

- **亲密度**（0-100）：聊得越多、越走心，亲密度越高
- **心情**（0-100）：她也会有好心情和坏心情
- **钱包余额**：可以向角色发虚拟转账红包，余额会真的记录在她的账户里
- **亲密度速率可调**：选择"慢热模式"，亲密度增长更慢，体验慢慢靠近的感觉

---

### 🃏 导入任意角色卡

支持从 **SillyTavern** 标准 PNG 角色卡导入：

- 拖入 PNG 文件，自动解析角色设定
- AI 智能提炼成格式统一的性格设定文件
- 导入前可以预览和手动调整
- 也可以从零开始，让 AI 引导你一步步创建一个全新角色

---

### 🔧 自由接入任何 AI 模型

Echo 不绑定任何特定的 AI 服务：

- **OpenAI、Anthropic Claude、Google Gemini**——主流大模型全支持
- **Ollama 本地大模型**——完全离线，零费用
- 任何兼容 OpenAI 格式的 API（DeepSeek 等）都能接
- 支持**主模型 + 辅助模型**双轨配置：后台记忆整理用便宜的小模型，聊天用强力的大模型，省钱又好用

---

### 🔒 数据完全属于你

Echo 的核心理念就是**本地优先，隐私第一**：

- 所有聊天记录、记忆、角色设定，全部存在你电脑的本地文件夹里
- 没有任何云端服务器，没有数据上传
- 角色的记忆文件是纯文本 Markdown，你可以随时打开来看、手动修改
- 支持一键导出 `.echo` 备份文件，可以完整迁移到另一台电脑

## 💡 一些让你想不到的细节

- **实时天气感知**：角色知道你所在城市的当前天气，会自然地聊到"下雨了你要带伞哦"
- **当前时间感知**：她知道现在是早上还是深夜，对话语气会自然切换
- **听歌共鸣**：开着音乐播放器和她聊天，她能感受到你在听什么歌，甚至跟着歌词产生情绪共鸣
- **聊天记录搜索**：全文搜索所有和某个角色的聊天历史
- **消息重新生成**：不满意上一条回复？点重新生成
- **聊天中止**：发现回复跑偏了？点停止，立刻中断
- **角色免打扰**：某个角色暂时不想被打扰？一键静音，她不会再主动来找你
- **角色置顶**：最喜欢的角色置顶到会话列表顶部
- **数据统计面板**：查看你和 AI 之间的调用次数、Token 消耗量、使用时长

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/andclear/project-echo.git
cd project-echo

# 安装依赖
npm install

# 开发模式启动
npm run dev
```

启动后，点击设置图标配置你的 AI API Key，然后创建或导入第一个角色，就可以开始了。

## 📂 你的数据放在哪里？

Echo 会在系统的用户数据目录下创建专属文件夹，结构非常清晰：

```
Echo 用户数据/
├── database/          # 聊天记录数据库
├── characters/
│   └── 角色名/        # 每个角色的专属目录
│       ├── Soul.md    # 性格设定
│       ├── Memory.md  # 记忆（LTM + STM）
│       ├── Diary.md   # 角色日记
│       ├── State.md   # 亲密度 / 心情 / 余额
│       ├── Schedule.md # 近7天日程
│       └── avatar.png # 头像
└── EchoMusicSources/  # 自定义音乐源脚本
```

所有文件都是可读的纯文本，你随时可以打开编辑。
